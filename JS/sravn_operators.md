# sravn_operators
---
Оператор `==` преобразует операторы разных типов к числу
```js
alert( 0 == false ); // true
alert( '' == false ); // true
```

Оператор `===` проверяет равенство без приведения типов
```js
alert( 0 === false ); // false
alert( '' === false ); // false
```
