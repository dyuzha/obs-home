# use_strict
---
```js
"use_strict";
// Этот код работает в современном режиме
```

```js
// Можно писать коментарии на надо этой строчкой
"use_strict";
// Этот код работает в современном режиме
```

```js
alert("some code");
// "use strict" ниже игнорируется - он должен быть в первой строке

"use strict";

// строгий режим не активирован
```

```js
// Для конкретной функции
(function() {
  'use strict';

  // ...ваш код...
})()

```
