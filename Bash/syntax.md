# Syntax
---

## Операторы сравнения
---


| Command | Action           |
|---------|------------------|
| -eq     | равно            |
| -ne     | не равно         |
| -gt     | больше           |
| -lt     | меньше           |
| -ge     | больше или равно |
| -le     | меньше или равно |
| -a      | And              |
| -o      | Or               |


>[!Example]
```
[ "$a" -le "$b" ]
```


| Command | Action           |
|---------|------------------|
| <       | меньше           |
| <=      | меньше           |
| >       | больше           |
| >=      | больше или равно |
| &&      | And              |

| `||`    | или              |


>[!Example]
```
((a" < "$b"))
```


## Конструкция CASE
---
```bash
fruit="apple"

case $fruit in
  "apple") echo "Это яблоко";;
  "banana") echo "Это банан";;
  *) echo "Неизвестный фрукт";;
esac
```


## Запуск скриптов с локальными переменными
---
```bash
# Запустить command, временно установив для нее переменную окружения VAR со значением value
VAR=value command
DEBUG=1 ./script.sh
```
