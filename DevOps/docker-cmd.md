##  Управление
---
```bash
# Чтобы принудительно обновить кеш
docker build --no-cache

# Пробрасывание порта
docker run -p 80:80

# Пробрасывание всех EXPOSE-портов
docker run -P

# для отладки, запуска дополнительных команд (например, проверка файлов, установка пакетов).
docker exec -it

# для мониторинга вывода основного процесса (логи) или взаимодействия с ним (если это интерактивная программа, например, python или bash).
docker attach
```
