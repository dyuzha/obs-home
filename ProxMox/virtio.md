# virtio
---
Фреймворк паравиртуализации

Эмуляция полного железа (медленно)
Паравиртуализация - техника виртуализации, при которой гостевая ОС подготавливается для использования в виртуальной среде
*(драйверы на гостевой ОС)*



Virtio как стандарт:

| Устройство             | Название                 | Note         |
|------------------------|--------------------------|--------------|
| Диски                  | virtio-blk / virtio-scsi |              |
| Сетевые устройства     | virtio-net               |              |
| Графические устройства | virtio-gpu               | Сырой        |
| Входные устройства     | virtio-input             |              |
| Консоль/serial         | virtio-serial            |              |
| RNG (генератор чисел)  | virtio-rng               | Криптография |
| Ballo0n (память)       | virtio-ballon            |              |
