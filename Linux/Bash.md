# Syntax
## Операторы сравнения
---
### Первый вариант
```bash
# И
-a  
&&	

# Или
-o 
||	

eq # равно
ne # не равно
gt # больше
lt # меньше
ge # больше или равно
le # меньше или равно
```

>[!Example]
>```
[ "$a" -le "$b" ]

### Второй вариант
```bash
- < 			– меньше 
- <= 			– меньше
- > 			– больше 
- >= 			– больше или равно 
```

>[!Example]
>```
((a" < "$b"))
