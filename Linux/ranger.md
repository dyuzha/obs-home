
>[!Note] CLI
```sh
<return> or l(right)   Открыть файл
```

>[!Note] **z**
```sh
zh                      показывает скрытые файлы
```

>[!Note] **y**
```sh
yy          Yank
    y.      Yank name (whith out extension)
    yn      Name
    yd      Dir
    yp      Puth
    ya      Скопировать mode add
    yr      Скопировать mode remove
    yv      Скопировать mode toggle
```

>[!Note] **p**
```sh
pp          Insert
```
>[!Note] **d**
```sh
dd          Cut
```
