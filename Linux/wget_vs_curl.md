# [[wget]] vs [[curl]]
---
  | Критерий                    | wget            | curl               |
  |-----------------------------|-----------------|--------------------|
  | Основное направление        | Загрузка файлов | API и HTTP-запросы |
  | Поддержка  рекурс. загрузки | Да              | Нет                |
  | Может работать в фоне       | Да (-b)         | Нет                |
  | Сохранение файлы            | Автоматически   | -o                 |
  | Наличие докачки             | (-с)            | Как-то сложно      |
